# 이전 버전 파일들

이 폴더에는 이전에 사용했던 Python 기반 파일들이 저장되어 있습니다.

## 파일 목록

- `api_server.py` - 이전 Python 기반 API 서버 (Flask 사용)
- `ai_content_generator.py` - AI 콘텐츠 생성기
- `openai_helper.py` - OpenAI API 헬퍼
- `qr_generator.py` - QR 코드 생성기
- `start.py` - 이전 시작 스크립트

## 현재 상태

현재 프로젝트는 Node.js 기반으로 전환되었으며, 다음 파일들이 사용되고 있습니다:

- `src/backend/api_server.js` - 현재 Node.js 기반 API 서버
- `src/database/services.js` - PostgreSQL 데이터베이스 서비스
- `admin/dashboard.html` - 관리자 대시보드
- `index.html` - 메인 페이지

## 주의사항

이 파일들은 참고용으로만 보관되어 있으며, 현재 프로젝트에서는 사용되지 않습니다.
필요시 참고하여 기능을 이식할 수 있습니다.
